Library Weasel

A breadboard for open data.

Library Weasel is a free software project with the goal of making it easier for people to work with open data in a variety of ways. It does this by leveraging a plugin system that allows for new data models and tools to be added incrementally and share common infrastructure. On top of this module system sits an embedded server running a modern and flexible application toolkit, Vert.x.
